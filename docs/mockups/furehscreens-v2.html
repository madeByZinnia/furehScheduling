02 Map — one shared building/floor control

No SITE pill: an empty selection is the site. Selecting a hotel reveals its floors (Main auto-selects); selecting a floor shows that floor. The same control filters both Map and List. Building-only zooms to the hotel's OSM outline (map) or lists all its rooms (list); floor-selected shows the drawn floorplan (map) or just that floor's rooms (list). OSM always renders outside the buildings — no runtime tiles.

04 Notes on the revisions

Why these changes, in one place.

One control, no SITE pill

The map's building/floor selector is the site control — an empty selection means "the whole site". It drives both Map and List: building-only shows the OSM outline / all that hotel's rooms; a floor shows the drawn plan / just that floor's rooms. One mental model, two views.

Text size is a guarantee, not a suggestion

Telegram's webview does not reliably forward the OS text setting (iOS WKWebView ignores Dynamic Type; the WebApp API exposes no font field). So the in-app control stays — now a discrete slider with a lower default (M), split from the theme toggle.

List view is the map in words

Not the old Now tab. A header per room, A→Z, each with its current panel and any crew present — with the signal named (GPS + star, star only) and how stale it is. At high magnification this is often faster than panning the canvas.

Glacier was a bad map, not a bad parse

The first PDF was wrong. The corrected QRG places Glacier on Delta 2nd between Canmore and Lethbridge, and Banff / Lake Louise / Jasper simply don't exist. All 21 scheduled rooms now resolve — the room gap is closed.

OSM outdoors, baked not fetched

Now that the map is our own vector, we bake OSM streets/parking/paths into the SVG at build time — zero runtime tile requests, one shared metric transform so GPS still lands exactly. The only cost is a visible © OpenStreetMap contributors credit.

Cancel belongs to the event, not a stray card

The standalone "cancel" card is gone. Edit reopens the create screen pre-filled; Cancel event sits at its bottom behind a "you sure" check, and shows [CANCELLED] to anyone who starred it.

Underground floors, anchored by their stairwells

Wyndham's Parking 1 & 2 are underground (confirmed with the hotel) — no OSM footprint of their own. But the elevator opposite the Front Desk and the bottom-right stairs appear on every level, so we align P1/P2 to the Main Floor by those shared icons as control points, then align the Main Floor to the street map. One coordinate system, three stacked floors — and it generalizes: Delta 2nd aligns to Delta Main the same way, so only one floor per building ever needs an OSM anchor.
